Run the Main function in order to generate data from a small set of raw images

Data folder contains two sets of data. 
First is raw data for beads. 
Second is raw data for different cell lines and empty positions. 
Main code files will run and save necessary data for later code files in processed data.

Main_Full will run the save full dataset inside the cells folder.
